# ORCA CSV Generator

Static HTML tool to parse ORCA quantum chemistry output (`.out` / `.log`) and export structured CSVs.

## Local usage
- Open `index.html` in any modern browser.
- Drop an ORCA output file to parse, preview data, and download CSV exports.

## Deploy to Netlify
- **Site type:** Static site (no build).
- **Publish directory:** `.` (repository root).
- **Build command:** _none_.
- **Optional:** Add a custom domain or enable drag-and-drop deploys from the Netlify UI if you prefer not to connect the repo.

## Deploy to GitHub Pages (optional)
- Push this repo to GitHub.
- Enable GitHub Pages for the `main` branch (root).
